Power BI Sales Analystics dashboard with DAX, time intelligence (YoY, YTD), and interactive insights.
# Sales Analytics Dashboard | Power BI Project

## Project Overview
This project focuses on analyzing sales performance using Power BI. The goal was to transform raw transactional data into meaningful business insights related to revenue, profit, customer behavior, and time-based trends.

## Tools & Technologies
- Power BI (Data Modeling, DAX, Visualizations)
- Power Query (Data cleaning and SQL-style transformations)
- Excel / CSV (Data source)

## Key Features
- Built a star schema data model with fact and dimension tables
- Created reusable DAX measures for KPIs and time intelligence
- Implemented YoY, YTD, and MoM analysis using a dedicated date table
- Designed a multi-page interactive dashboard

## KPIs & Metrics
- Total Sales
- Total Profit
- Profit Margin %
- Total Orders
- Average Order Value (AOV)
- YoY Growth %
- Sales YTD

## Dashboard Pages
1. Executive Summary – high-level KPIs and trends
2. Sales & Profit Trends – YoY, YTD, monthly analysis
3. Customer Insights – top customers and order behavior
4. Product & Payment Analysis – category and payment mode insights

## Key Insights
- Identified top 10 customers contributing the highest revenue
- Analyzed category-level profitability
- Observed seasonal trends using YoY and YTD metrics
